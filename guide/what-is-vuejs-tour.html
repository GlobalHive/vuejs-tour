# What is VueJS Tour

VueJS Tour is a component for creating guided tours. It's written in Vue 3 composition api and uses the latest version of PopperJSopen in new window

# Features

  • Simple API
  • Customizable theme
  • Written in Vue 3 composition api
  • onNext() and onPrev() callbacks
  • Scoped slots for customizing content and buttons

WARNING

VueJS Tour is written for Vue 3 composition api. There are no plans to support Vue 2.x
